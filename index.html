Steganography.js
2017

A mobile steganography app.

Create, analyze and compare steganographic images online or offline! Steganography allows you to conceal data (images, text) in another digital file.

  • Open and analyze any image to find hidden images or text,
  • download the extracted hidden image,
  • compare two images to see if any of them have been tampered with or have hidden data,
  • see live statistics of the app worldwide.

The app uses several different Web Workers for better performance and a smooth experience. The user can also install the app on their device as a Progressive Web App to use it while offline.
